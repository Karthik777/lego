This is a sample teaching post. Each Sunday's video, reflection, or scripture post lives here as its own Markdown
file in `lego/church/teachings/`.

To publish a YouTube video, add a fenced code block with the language set to `youtube`:

```youtube
https://www.youtube.com/watch?v=dQw4w9WgXcQ
```

To publish a Google Drive video, share the file with "Anyone with the link" and use a `gdrive` fenced code block
with the share URL (the `.../file/d/FILE_ID/view...` link):

```gdrive
https://drive.google.com/file/d/1a2b3c4d5e6f7g8h9i0j/view?usp=sharing
```

Write your reflection or scripture notes underneath the video as regular Markdown. Delete this file once you've
published your first real teaching.
